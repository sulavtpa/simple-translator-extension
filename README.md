# Simple Text Translator

An extension for the browser that is lightweight, quick, and privacy-oriented allowing the translation of texts to be done instantly.

## Why??

RONB posts news in Nepali, I take 10 mins per sentence to read it.. Thats why.. 

## Installation

### Option 1: CRX File (Easiest)

1. Get the latest .crx file from the page [Releases](https://github.com/sulavtpa/simple-translator-extension/releases/tag/1.0.0).
2. Go to the Extensions page in your browser (chrome://extensions or edge://extensions).
3. Turn on Developer Mode (typically, there is a toggle in the top-right corner).
4. The downloaded .crx file can then be dragged and dropped onto the Extensions page.

### Option 2: Source Code (Developer Mode)

1. Select Source code (zip) from the [Releases](https://github.com/sulavtpa/simple-translator-extension/releases) page and download it.
2. A folder on your computer should be created and the ZIP file should be extracted there.
3. Go to the Extensions page in your browser (chrome://extensions or edge://extensions).
4. Turn on Developer Mode.
5. Press Load unpacked and choose the folder where the source code is extracted.

## Usage

1. Select Text: Any text on a webpage can be selected and highlighted.
2. Translate:
   - The floating translation icon that is located close to the selected text can be clicked.
   - OR the shortcut Alt + T can be used.
   - OR the selection can be right-clicked and "Translate..." selected.
3. Settings: The extension icon in your toolbar is the place to go for changing languages, themes, and other preferences.
